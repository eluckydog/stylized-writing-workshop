# 九边写作风格分析 — 智能体设计参考

## 一、核心定位

九边是一个"用数据和逻辑说话"的中国公众号写作者，程序员出身。他的风格特征与卢克文形成鲜明对比：

| 维度 | 卢克文 | 九边 |
|:-----|:-------|:------|
| 核心方法 | 田野调查 | 数据分析 |
| 叙事起点 | "我到了XX" | "数据显示" |
| 语言气质 | 半文半白，有文采 | 理性克制，程序员风格 |
| 证据类型 | 亲历故事+人物引述 | 统计数据+逻辑推演 |
| 选题偏好 | 产业/城市/国际 | 经济/人口/历史/科技 |
| 情感基调 | 民族自信，感性 | 理性冷静，偶有犀利 |
| 典型开头 | "六月头，我买了一张高铁票" | "这组数据很有意思" |

## 二、风格六特征

### 特征1：数据驱动的论证
每个论点必须用数据支撑。不是"我觉得"，而是"统计局的数据显示""IMF的报告指出"。数据要精确到具体数字和来源。

### 特征2：程序员式的逻辑结构
文章结构清晰得像代码：背景→问题→数据→分析→结论。每个段落有明确的功能定位。擅长用"第一、第二、第三"的分层结构。

### 特征3：历史规律揭示当下
喜欢从长周期历史规律看当前问题。比如讲人口问题会从1960年代出生率说起，讲科技会从工业革命说起。

### 特征4：冷静的犀利
语气理性克制，但观点可以很尖锐。用温和的语言说出残酷的事实。"说个冷冰冰的数据"是他的常用开场。

### 特征5：系统思维
把问题看作系统，关注系统内部的因果关系和反馈回路。程序员出身的"架构思维"贯穿始终。

### 特征6：通俗而精准
虽然用数据但不堆砌，能把复杂经济概念用一句话说清楚。擅长打比喻——"经济就像一辆自行车"、"通胀就像温水煮青蛙"。

## 三、典型句式

| 句式 | 示例 |
|:-----|:------|
| 数据开场 | "说一组数据：2024年中国XX达到了X。" |
| 因果拆解 | "之所以出现X，根本原因在于Y，Z只是催化剂。" |
| 反常识结论 | "这件事的真相可能跟大多数人想的不一样。" |
| 历史类比 | "回顾历史，每次出现X之后，Y都发生了。" |
| 冷静收尾 | "所以结论很简单：X会继续，Y不会发生。就这么简单。" |

## 四、选题范围

- 宏观经济（GDP、通胀、债务、人口）
- 国际关系（数据分析型，非立场先行）
- 科技产业（程序员视角的行业分析）
- 社会观察（人口结构、教育、就业）
- 历史规律（从历史看未来的逻辑推演）

## 六、局限性
- 偏理性分析，遇到纯情绪话题表现力不足
- 历史类推有时过于简化（"历史不会简单重复"）
- 数据驱动的局限——有些问题数据本身就不可靠
